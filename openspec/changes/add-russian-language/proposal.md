# Добавление поддержки русского языка

## Контекст
PDFCraft уже имеет файл перевода для русского языка (`messages/ru.json`), но русский язык не интегрирован в систему локализации. Необходимо полностью интегрировать русский язык в приложение, чтобы пользователи могли выбрать русский язык интерфейса.

## Цель
Полностью интегрировать русский язык в систему локализации PDFCraft, чтобы обеспечить полноценную поддержку русскоязычного интерфейса.

## Требования
- Русский язык должен быть доступен в списке языков
- Интерфейс должен корректно отображаться на русском языке
- Все текстовые элементы должны быть переведены
- Выбор русского языка должен сохраняться между сессиями
- Русский язык должен использовать правильные форматы дат и чисел

## Ограничения
- Все операции локализации должны происходить на клиенте
- Не должно быть дублирования кода
- Производительность не должна ухудшиться
- Русский язык должен использовать направление текста слева направо (LTR)
