# Спецификация поддержки русского языка

## ADDED Требование: Поддержка русского языка в интерфейсе

### Сценарий: Пользователь выбирает русский язык
Когда пользователь открывает выпадающий список выбора языка
И пользователь выбирает "Русский" (Русский)
Тогда интерфейс приложения должен переключиться на русский язык
И все текстовые элементы должны отображаться на русском языке
И URL должен обновиться с добавлением префикса "/ru"

## ADDED Требование: Сохранение предпочтений пользователя

### Сценарий: Сохранение выбора русского языка
Когда пользователь выбирает русский язык
Тогда предпочтение должно сохраняться в локальном хранилище браузера
И при следующем посещении сайта интерфейс должен автоматически отображаться на русском языке

## ADDED Требование: Корректное отображение дат и чисел

### Сценарий: Отображение дат на русском языке
Когда интерфейс отображается на русском языке
Тогда даты должны отображаться в формате DD.MM.YYYY
И числа должны отображаться с использованием русской локали

## ADDED Требование: Поддержка направления текста

### Сценарий: Направление текста для русского языка
Когда интерфейс отображается на русском языке
Тогда текст должен отображаться слева направо (LTR)
И не должно быть изменений в направлении потока элементов интерфейса
