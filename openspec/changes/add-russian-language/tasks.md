# Задачи для интеграции русского языка

## Подзадачи

1. Обновить конфигурацию локалей
   - Добавить 'ru' в массив поддерживаемых локалей
   - Добавить информацию о русском языке (название, родное название, направление текста, формат даты)

2. Обновить функцию генерации локализованных путей
   - Добавить 'ru' в список локалей, используемых при генерации путей

3. Обновить компонент выбора языка
   - Добавить русский язык в выпадающий список выбора языка

4. Проверить и протестировать
   - Убедиться, что русский язык корректно отображается в интерфейсе
   - Проверить, что переводы загружаются правильно
   - Проверить, что выбор языка сохраняется между сессиями
