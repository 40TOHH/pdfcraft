# Change Proposal: Добавление переключателя темной темы

## Контекст

В настоящее время сайт PDFCraft имеет частично реализованную темную тему в CSS (в файле `globals.css`), но отсутствует пользовательский интерфейс для переключения между светлой и темной темами. Пользователи могут видеть темную тему только если их системные настройки предполагают использование темной темы по умолчанию.

## Проблема

Отсутствие возможности для пользователя вручную переключаться между светлой и темной темами. Это ограничивает доступность и удобство использования сайта, особенно в различных условиях освещения.

## Решение

Добавить компонент переключателя темы (Theme Toggle Switch), который позволит пользователям выбирать предпочитаемую тему независимо от системных настроек. Компонент будет:

- Сохранять выбор пользователя в localStorage
- Применять соответствующий CSS класс (`.dark`) к элементу `<html>`
- Сохранять выбор между сессиями
- Интегрироваться в существующий интерфейс (например, в шапку сайта)

## Влияние

- Повышение доступности и удобства использования
- Улучшение пользовательского опыта в разных условиях освещения
- Совместимость с существующей CSS-системой тем
- Соответствие современным веб-стандартам

## Риски

- Необходимость тестирования в разных браузерах
- Потенциальные проблемы с гидратацией в Next.js приложениях
- Необходимость обеспечения согласованности темы по всему приложению
