# Спецификация: Изменение основных цветов интерфейса

## ADDED Requirements

### Requirement: Основной цвет (Primary Color)
- **Описание**: Система SHALL использовать хурму (#FF5C34) в качестве основного брендового цвета
- **Значение**: `--color-primary: 12 100% 59%` (HSL для #FF5C34)
- **Применение**: Используется для основных кнопок, ссылок и других важных интерактивных элементов

#### Scenario: Отображение основной кнопки
- При рендере компонента основной кнопки
- Система применяет новый цвет к фону кнопки
- Ожидаемый результат: кнопка отображается с цветом хурмы (#FF5C34)

## MODIFIED Requirements

### Requirement: Цвет текста на основном фоне
- **Описание**: Система SHALL обеспечивать достаточный контраст для текста на основном фоне
- **Значение**: `--color-primary-foreground: 0 0% 100%` (белый цвет)
- **Применение**: Используется для текста на элементах с основным цветом фона

#### Scenario: Отображение текста на основной кнопке
- При рендере текста внутри основной кнопки
- Система применяет белый цвет к тексту
- Ожидаемый результат: текст четко виден на фоне хурмы

### Requirement: Цвет наведения на основной элемент
- **Описание**: Система SHALL использовать более темный оттенок основного цвета при наведении
- **Значение**: `--color-primary-hover: 12 100% 49%` (темнее на 10% по светлоте)
- **Применение**: Используется для визуальной обратной связи при наведении курсора

#### Scenario: Наведение на основную кнопку
- При наведении курсора на основную кнопку
- Система меняет цвет фона на темно-хурму
- Ожидаемый результат: пользователь получает визуальную обратную связь

### Requirement: Вторичный цвет
- **Описание**: Система SHALL использовать светло-хурмовый оттенок в качестве вторичного цвета
- **Значение**: `--color-secondary: 12 50% 85%` (светло-хурмовый оттенок)
- **Применение**: Используется для менее важных элементов интерфейса

#### Scenario: Отображение вторичной кнопки
- При рендере компонента вторичной кнопки
- Система применяет светло-хурмовый цвет
- Ожидаемый результат: кнопка выглядит гармонично с основной палитрой

### Requirement: Акцентный цвет
- **Описание**: Система SHALL использовать хурму (#FF5C34) в качестве акцентного цвета
- **Значение**: `--color-accent: 12 100% 59%` (тот же, что и основной цвет)
- **Применение**: Используется для выделения важной информации

#### Scenario: Отображение акцентированного текста
- При рендере текста с акцентом
- Система применяет хурмовый цвет к тексту
- Ожидаемый результат: текст выделяется на странице

### Requirement: Основной цвет в темной теме
- **Описание**: Система SHALL адаптировать основной цвет для темной темы
- **Значение**: `--color-primary: 12 100% 65%` (светлее на 5% по светлоте)
- **Применение**: Используется для основных элементов в темной теме

#### Scenario: Отображение основной кнопки в темной теме
- При активной темной теме
- Система применяет более светлый хурмовый цвет
- Ожидаемый результат: кнопка хорошо видна на темном фоне

### Requirement: Фоновый цвет в темной теме
- **Описание**: Система SHALL использовать темно-серый цвет в качестве фона в темной теме
- **Значение**: `--color-background: 222 47% 11%` (темно-серый)
- **Применение**: Используется как основной фон в темной теме

#### Scenario: Отображение страницы в темной теме
- При активной темной теме
- Система применяет темный фон ко всей странице
- Ожидаемый результат: контент читаем и не утомляет глаза
