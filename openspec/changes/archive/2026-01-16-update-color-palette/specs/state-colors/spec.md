# Спецификация: Изменение цветов состояний

## MODIFIED Requirements

### Requirement: Цвет успеха (Success Color)
- **Описание**: Система SHALL использовать зеленый оттенок (#34D399) для обозначения успешных действий
- **Значение**: `--color-success: 142 76% 36%` (зеленый оттенок, сохраняем яркость)
- **Применение**: Используется для сообщений об успешных операциях, галочек и т.п.

#### Scenario: Отображение сообщения об успешной операции
- При отображении уведомления о завершении операции
- Система применяет зеленый цвет к уведомлению
- Ожидаемый результат: пользователь видит, что операция прошла успешно

### Requirement: Цвет предупреждения (Warning Color)
- **Описание**: Система SHALL использовать оранжевый оттенок (#F59E0B) для обозначения предупреждений
- **Значение**: `--color-warning: 45 100% 51%` (оранжевый оттенок, #F59E0B)
- **Применение**: Используется для предупреждающих сообщений и элементов

#### Scenario: Отображение предупреждающего сообщения
- При отображении уведомления с предупреждением
- Система применяет оранжевый цвет к уведомлению
- Ожидаемый результат: пользователь обращает внимание на потенциальную проблему

### Requirement: Цвет ошибки (Destructive Color)
- **Описание**: Система SHALL использовать красный оттенок для обозначения ошибок и разрушительных действий
- **Значение**: `--color-destructive: 0 72% 51%` (красный оттенок)
- **Применение**: Используется для сообщений об ошибках, кнопок удаления и т.п.

#### Scenario: Отображение сообщения об ошибке
- При возникновении ошибки валидации формы
- Система применяет красный цвет к сообщению об ошибке
- Ожидаемый результат: пользователь видит, что произошла ошибка

### Requirement: Цвет фокуса (Ring Color)
- **Описание**: Система SHALL использовать хурму (#FF5C34) в качестве цвета обводки при фокусировке
- **Значение**: `--color-ring: 12 100% 59%` (тот же, что и основной цвет)
- **Применение**: Используется для визуализации фокуса клавиатуры

#### Scenario: Перемещение по элементам с клавиатуры
- При навигации с клавиатуры между интерактивными элементами
- Система показывает кольцо фокуса вокруг активного элемента
- Ожидаемый результат: пользователь видит, какой элемент сейчас активен
