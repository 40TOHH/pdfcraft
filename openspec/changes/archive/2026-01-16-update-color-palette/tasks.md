# Задачи: Обновление цветовой палитры

## Порядок выполнения

1. Обновить CSS переменные в globals.css
2. Обновить цвета для темной темы
3. Проверить все компоненты на предмет правильного отображения новых цветов
4. Обновить цвета состояний (успех, ошибка, предупреждение)
5. Провести тестирование доступности
6. Обновить документацию (если применимо)

## Подробные задачи

### 1. Обновление CSS переменных
- [ ] Заменить значения переменных основных цветов в :root
- [ ] Обновить значения переменных для темной темы
- [ ] Проверить, что все переменные используют HSL формат

### 2. Обновление цветов темной темы
- [ ] Обновить значения переменных в блоке .dark
- [ ] Проверить контрастность всех элементов в темной теме
- [ ] Убедиться, что цвета остаются визуально согласованными

### 3. Проверка компонентов
- [ ] Протестировать основные кнопки (primary, secondary, accent)
- [ ] Проверить цвета заголовков и основного текста
- [ ] Проверить цвета карточек и границ
- [ ] Проверить цвета уведомлений и модальных окон

### 4. Обновление цветов состояний
- [ ] Обновить цвета успеха, ошибки и предупреждения
- [ ] Проверить, что они хорошо контрастируют с новой палитрой
- [ ] Убедиться, что цвета соответствуют требованиям доступности

### 5. Тестирование доступности
- [ ] Проверить соотношение контрастности текста и фона (должно быть ≥ 4.5:1)
- [ ] Проверить контрастность интерактивных элементов
- [ ] Убедиться, что цвета фокуса видны на всех фонах
- [ ] Проверить, что информация не передается только через цвет

### 6. Обновление документации
- [ ] Обновить документацию по стилям (если существует)
- [ ] Зафиксировать новые значения цветов в справочнике компонентов (если есть)
- [ ] Обновить примеры использования цветов в документации

## Зависимости

- Задача 3 зависит от завершения задач 1 и 2
- Задача 5 должна выполняться после всех обновлений цветов
- Задача 6 может выполняться параллельно с другими задачами

## Критерии завершения

- Все цвета обновлены в соответствии со спецификацией
- Сохранена доступность интерфейса
- Все компоненты корректно отображаются с новыми цветами
- Предоставлены доказательства тестирования доступности
