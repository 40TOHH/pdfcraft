# Design Document: Обновление цветовой палитры

## Архитектурные решения

### Цветовая модель

Для реализации нового дизайна будет использована HSL (Hue, Saturation, Lightness) модель, которая уже используется в текущем проекте. Новый основной цвет хурма (#FF5C34) будет представлен в HSL как:

- Hue: 12°
- Saturation: 100%
- Lightness: 59%

### Структура CSS переменных

Будут обновлены CSS переменные в файле `globals.css`, сохраняя текущую структуру именования для обеспечения совместимости с существующими компонентами.

### Темная тема

Для темной темы будут подобраны соответствующие контрастные оттенки, сохраняющие визуальную идентичность бренда.

## Компоненты, требующие обновления

### Основные цвета
- `--color-primary`: будет изменен на хурму (#FF5C34)
- `--color-primary-foreground`: останется белым для контраста
- `--color-primary-hover`: будет темнее основного цвета для визуальной обратной связи

### Второстепенные цвета
- `--color-secondary`: будет адаптирован для гармоничного сочетания с основной палитрой
- `--color-accent`: может остаться без изменений или быть заменен на дополнительный оттенок

### Фоновые цвета
- `--color-background`: останется белым
- `--color-foreground`: будет скорректирован для обеспечения доступности

### Цвета состояний
- `--color-success`, `--color-warning`, `--color-destructive`: будут пересмотрены для обеспечения контраста с новой палитрой

## Доступность

Все новые цветовые комбинации будут протестированы на соответствие стандартам WCAG 2.1 AA для обеспечения доступности.

## Темная тема

Для темной темы будут созданы соответствующие оттенки, сохраняющие визуальную идентичность:

- Основной цвет будет заменен на соответствующий темный оттенок хурмы
- Фоновые цвета будут затемнены
- Текстовые цвета будут осветлены для обеспечения читаемости
